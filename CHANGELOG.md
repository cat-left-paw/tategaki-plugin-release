## [1.2.7] - 2026-03-20

### Changed

- 表示設定の見出し設定・フロントマター設定・選択設定まわりのUI文言と既定値を見直し、設定内容が分かりやすくなるよう整理
- ユーザー向け名称を見直し、`SoT` 表記を `執筆・参照` 系の表記へ統一
- 互換モード / 書籍モード / 設定まわりの内部実装と公開ドキュメントを整理し、コミュニティプラグイン審査向けの整合性を改善

### Fixed

- 表示設定で見出し色をリセットした際に、見出し設定セクションが閉じてしまう問題を修正
- 表示設定で書字方向を変えた後、見出し位置の表示が変更前の書字方向に引きずられる問題を修正
- 書籍モードで見出しを章扉扱いにした場合、フロントマター表示方式が不自然になる問題を改善し、章扉時は独立ページ扱いになるよう整理
- 書籍モードで見出しページの扱いを `なし` にしていても改ページされてしまう問題を修正
- フロントマターを本文先頭に表示する設定で、直後の改ページ見出し由来の下線が残ることがある問題を修正
- ツールバーの書字方向切り替えボタンの hover 表示を改善し、他のボタンでチラつきが出る問題を修正
- 書籍モードで、見出し設定のフォントと文字色が見出し表示に反映されない問題を修正

## [1.2.6] - 2026-03-14

### Fixed

- 執筆・参照モードで見出しを折りたたんだ状態の大きい文書を編集した際、入力確定時にスクロール位置が大きく揺れたり、右上方向へフラッシュしたりする問題を修正
- 執筆・参照モードで見出しの折りたたみ切り替え時に、表示が不安定になりやすい問題を改善
- 折りたたみ見出しを含む差分更新処理を見直し、hidden 行と collapsed gap の整合性を改善

## [1.2.5] - 2026-03-13

### Added

- 書籍モードに、フロントマターを本文から分離して独立ページとして表示する設定を追加
- 書籍モードに、見出し前の改ページと章扉ページ挿入の設定を追加
- 表示設定に、見出し後マージン、見出し区切り線、見出し位置の調整を追加
- 執筆・参照モードに選択モード設定を追加（クリック優先 / ドラッグ選択優先）
- 表示設定モーダルから Obsidian のプラグイン設定を直接開く導線を追加

### Changed

- 書籍モードのページ分割ロジックを見直し、表紙ページ・章扉ページ・空要素の扱いを改善
- 書籍モードで表紙ページを使う場合、表紙を `0` ページとして扱い、本文ページ数を分かりやすく表示するよう改善
- ユーザー向けの CE補助モード切り替え UI を廃止し、執筆・参照モードの選択・入力まわりの挙動を整理
- 執筆・参照モードの段落単位ソーステキスト編集モードを改良し、表示との連携や操作感を改善
- 執筆・参照モード / 互換モードのリスト・引用ブロックの Enter 挙動を見直し、継続入力や終了条件を改善
- リスト項目内や引用ブロック内で `Shift+Enter` による項目内改行に対応
- `Shift+Enter` で挿入した空行を、そのまま継続入力できるよう改善
- フロントマター解析を執筆・参照モード / 互換モード / 読書ビューで共通化し、YAML の解釈を改善
- リボンや起動時のビュー選択で、読書モードも前回の選択状態として記憶するよう改善
- IME 表示補正スライダーを、既定値基準の相対表示 `-1..+1` に整理
- 組版設定を見直し、`text-justify: inter-character`、`text-wrap: pretty`、執筆・参照モードの縦書き時の `line-break: strict` を適用

### Fixed

- Tategaki タブを閉じる際に、別ノート側のフォーカスを奪ってしまう問題を修正
- フロントマター付きファイル初回表示時に、本文先頭より不自然にスクロールされる問題を修正
- 執筆・参照モードの大きい文書で、スクロール停止後の可視領域復元が遅い問題を改善
- ルビ選択まわりの補助処理を整理し、執筆・参照モード / 互換モード双方で選択操作の安定性を改善

## [1.2.4] - 2026-02-11

### Added

- 執筆・参照モードのツールバーにチェックリストボタンを追加（`- [ ]` / `- [x]` の切り替え）
- 縦中横（TCY）を明示指定できるトグル操作を強化（ツールバー/右クリックメニューを統一）
- 縦中横の独自記法 `｟...｠` を導入し、明示指定した範囲のみTCY化できるように対応
- 自動縦中横オプションを追加（英数字2〜4文字 + `!!` / `!?` / `??` を表示時のみTCY化）
- ルビ挿入で傍点候補の拡張（候補追加/削除、カスタム候補の保存）

### Changed

- UI文言の英語対応範囲を拡張（通知、モーダル、検索置換、補助入力、アウトライン、書籍/互換モード周辺）
- 執筆・参照モードのリスト項目移動処理をCE補助モード対応のため整理し、関連ロジックを分離
- CE補助モード（Overlay IME）で装飾済みテキストを編集する際、入力中テキストが元の装飾スタイルに追従するよう改善

### Fixed

- 執筆・参照モードで「書式クリア」時にルビ記法が残るケースを修正
- CE補助モード時にリスト項目移動ショートカットが効かない問題を修正（IME変換中は安全のため無効）

### Notes

- TCYは2系統で利用可能
    - 明示TCY: `｟...｠`（独自記法）で、意図した箇所だけを固定的にTCY化
    - 自動TCY: 表示時のみ英数字2〜4文字と `!!` / `!?` / `??` をTCY化（本文に記号を追加しない）
- 明示TCYの `｟...｠` は独自記法のため、他エディタや未対応環境では記号がそのまま見える場合があります

## [1.2.0] - 2026-01-29

### Added

- **（新）執筆・参照モード（WYSIWYG）を追加**
    - Obsidian の Markdown を直接編集する新方式（変換・同期処理が不要）
    - ペアノートをアクティブにすると、（新）執筆・参照モードのビューが参照表示として追従
    - 見出しの折りたたみ機能
    - 見出し（アウトライン）ジャンプ機能
    - 設定パネルからフォント・余白などを手軽に調整可能
    - 同期バックアップ不要（同期リスクなし）
    - ペアノート方式（Obsidian のノートと縦書きビューが連動）
- （新）執筆・参照モードの上下余白設定を設定パネルから調整可能に
- 互換モード: 同期バックアップの作成をON/OFFできる設定を追加
- 互換モード: 同期バックアップフォルダを開くボタンを追加

### Changed

- TipTap 版の執筆モードを「互換モード」に変更
    - v1.1.1 以前の執筆モード（変換・同期あり）として引き続き利用可能
    - 当面維持
- モバイル非対応に変更（`manifest.json` の `isDesktopOnly: true`）
- プラグイン設定・表示設定の整理
- 表示設定の項目を追加（IME表示補正、キャレット設定、書籍モードのヘッダー/フッターなど）

### Removed

- コマンドパレットから以下のコマンドを削除
    - 執筆・参照を切り替え
    - 書字方向を切り替え（縦書き・横書き）
    - 書籍モードビューを開く・閉じる
    - 直前の同期バックアップを開く・復元
- 配布Zipへの OS別インストーラースクリプト同梱を廃止（同梱は v1.1.1 まで）

### Fixed

- 書籍モード: ページを跨ぐ際にルビつきの文字があるとレイアウトが乱れる問題を修正

## [1.1.0] - 2026-01-03

### Changed

- 同期バックアップ（自動バックアップ）を大幅に見直し
    - 以前: 同期のたびに無条件で作成し、最新 5 ペアを超えると古いものから削除
    - 変更後: セッション開始時/手動同期/一定間隔（例: 20 分、dirty 時のみ）/大量ペースト/危険な変更（ルビ・装飾など）といった理由に応じて作成し、保持も「使用日ベース」などの方針で整理

### Added

- モード追加: 執筆モード(v1.0.0 公開時の WYSIWYG 編集機能) / 参照モード(新設) / 書籍モード(新設)
- 参照モード: Obsidian 側の編集内容をほぼリアルタイムに参照できる表示（同期や変換を最小化）
- 参照モード: Obsidian の Markdown を直接扱うため、変換や同期を伴う執筆モードより安全な編集用ワークフローを提供
- 書籍モード: 高速な実測ベースのページ分割（長文でも操作性を維持するための時間スライス処理）
- 書籍モード: アウトライン表示・見出しジャンプ（ページ生成に合わせて追記）
- 書籍モード: ページ遷移効果（フェード等）
- テーマ: プリセットテーマ追加（アッシュベリー（ライト）、アッシュベリー（ダーク）、ダスティネイビー）

### Fixed

- その他: UI などの細かい修正

### Known Issues

- 書籍モード: 新しいウィンドウ（独立したウィンドウ）で開いた場合、マウスホイールでのページ移動ができません（方向キー / PageUp / PageDown は利用できます）。次回以降のアップデートで修正予定です。
- 書籍モード: ページ分割中にマウスホイールを操作すると、ちらつきが発生する場合があります（次回以降のアップデートで改善予定）。
