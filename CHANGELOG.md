# Changelog

このプロジェクトの主な変更点をまとめます。

形式は Keep a Changelog に概ね準拠し、バージョンは SemVer に従います。

## [1.3.0] - 2026-05-05

### Added

- 執筆・参照モードに、入力位置が画面内の一定位置に収まるよう自動でスクロールする「Typewriter scroll」を追加
    - 追従する位置と、スクロールを開始する範囲を調整可能
- 執筆・参照モードに、文書の末尾付近でも入力位置を見やすい高さに保てる余白設定を追加
- 執筆・参照モードに、現在の編集箇所へ集中しやすくする視覚フォーカス機能を追加
    - 編集中のブロックを強調表示
    - 現在行を強調表示
    - 編集箇所以外の段落 / ブロックを控えめに表示
- ツールバーに `Typewriter` メニューボタンを追加し、`Typewriter scroll`、編集ブロックの強調、現在行の強調、非フォーカス部分の減光をまとめて切り替え可能に
- 自動縦中横に「数字だけを対象にする」オプションを追加
    - ON のとき、通常の自動TCY対象を数字列のみに制限
    - `!!` / `!?` / `??` は例外として継続対応

### Changed

- 執筆・参照モードの `Home` / `End` の挙動を見直し、現在位置から表示行の先頭 / 末尾へ自然に移動できるよう改善
- 執筆・参照モードの `PageUp` / `PageDown` の挙動を見直し、画面単位で移動したあともキャレット位置が分かりやすく保たれるよう改善
- 執筆・参照モードの `End` キーの挙動を整理し、折り返し行の末尾付近でも安定して移動できるよう改善
- 執筆・参照モードの方向キー移動を見直し、折り返し行、装飾の境界、ルビ周辺でのキャレット移動をより安定化
- 執筆・参照モードの選択モードの新規既定値を「クリック優先」に変更
- Typewriter scroll 使用中は選択モードを実質的に「クリック優先」として扱い、設定画面でもその状態が分かるよう整理
- Typewriter scroll 関連の設定は、ソース表示、全文プレーン表示、段落単位ソース編集の使用中は一時的に利用不可として表示し、通常表示に戻ったときに再適用されるよう整理
- 視覚フォーカス機能は、Typewriter scroll を使わない場合でも単独で利用できるよう整理
- Typewriter scroll と視覚フォーカスに合わせて、表示設定、プラグイン設定、ユーザー向け文書を更新
- `Typewriter mode` のユーザー向け表記を `Typewriter scroll` に整理

### Fixed

- 執筆・参照モードで、折り返し行の `Home` / `End` / `PageUp` / `PageDown` が期待通りに動かない問題を修正
- 執筆・参照モードで、`End` キーを押したときの二段階移動が不安定になる問題を修正
- 執筆・参照モードで、Typewriter scroll 使用中に通常文字入力へスクロールが追従しないことがある問題を修正
- 執筆・参照モードで、現在行の強調表示が折り返し位置に正しく追従しない問題を修正
- 執筆・参照モードで、文書末尾の余白表示中に選択表示やクリック位置の補正が不安定になる問題を修正
- 執筆・参照モードで、水平線行にキャレットを置けず、`Delete` / `Backspace` で削除しづらい問題を修正
- 大きな文書で表示設定を開いたり設定を変更した際に、背後の本文が大きくスクロールしやすい問題を一部改善

## [1.2.4] - 2026-02-11

### Added

- 自動縦中横オプションを追加（英数字2〜4文字 + `!!` / `!?` / `??` を表示時のみTCY化）

### Changed

- UI文言の英語対応範囲を拡張（通知、モーダル、検索置換、補助入力、アウトライン、書籍/互換モード周辺）

### Notes

- TCYは2系統で利用可能
    - 明示TCY: `｟...｠`（独自記法）で、意図した箇所だけを固定的にTCY化
    - 自動TCY: 表示時のみ英数字2〜4文字と `!!` / `!?` / `??` をTCY化（本文に記号を追加しない）
- 明示TCYの `｟...｠` は独自記法のため、他エディタや未対応環境では記号がそのまま見える場合があります

- 書籍モード: 高速な実測ベースのページ分割（長文でも操作性を維持するための時間スライス処理）
- 書籍モード: アウトライン表示・見出しジャンプ（ページ生成に合わせて追記）
- 書籍モード: ページ遷移効果（フェード等）
